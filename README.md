# 💜 Broken But Beautiful | For Tanha

A bespoke, emotionally-driven web experience designed as a digital letter. This project blends soulful music, artistic typography, and immersive visual effects to create a "glass-memory" atmosphere.

## ✨ Features

* **Dynamic Background:** A blended image with a slow parallax pan effect and a pulsing purple aura.
* **Dual Language Support:** Instant toggle between a curated English "Love Theme" cursive font and an artistic "Bengali" script.
* **Immersive Audio:** Integrated YouTube API to play background mashups seamlessly upon user interaction.
* **Interactive Cursor:** A custom 3rd-layer heart cursor that leaves a trailing path of floating purple heart particles.
* **Glassmorphism UI:** A modern, translucent letter card built with high-end blur filters and responsive design.

---

## 🚀 Tech Stack

* **Frontend:** HTML5, CSS3 (Custom Variables, Keyframe Animations)
* **Interactions:** Vanilla JavaScript
* **Typography:** Google Fonts (Dancing Script, Maishan, Playfair Display)
* **Media:** YouTube IFrame API & Imgur hosting

## 🛠️ Setup & Usage

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/yourusername/broken-but-beautiful.git](https://github.com/yourusername/broken-but-beautiful.git)
    ```
2.  **Open the Project:**
    Simply open the `index.html` file in any modern web browser.

3.  **Interaction:**
    Click anywhere on the page to trigger the background music and start the emotional journey.

---

## 🎨 Design Philosophy

The project uses a dark, moody color palette centered around **#9d4edd** (Accent Purple) and **#020205** (Deep Onyx). The goal was to simulate a "fading memory" where the text remains the only clear constant.

> "Whether these calls are our way of healing or a bridge leading us back to one another... you are still my favorite thought."
